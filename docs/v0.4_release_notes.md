# v0.4 Release Notes

## Highlights

- Added leaderboard-ready run artifacts: `results.json`, `model_card.json`, `leaderboard_row.json`, and `leaderboard.csv`.
- Added v0.4 scoring dimensions for category, difficulty, calibration, efficiency, and trap avoidance.
- Added two stronger public baselines: `risk-aware` and `oracle-lite`.
- Promoted the benchmark version to `0.4.0`.
- Documented the v0.4 benchmark protocol and leaderboard schema.

## Compatibility

The audited 14-scenario MVP pack remains the stable public compatibility suite. v0.4 keeps the existing replay, audit, and publish-check commands intact.

## Recommended release verification

```bash
python -m lexcapital validate scenarios/mvp
python -m lexcapital run-baseline --policy risk-aware --scenarios scenarios/mvp --out runs/baselines/risk_aware_v0.4
python -m lexcapital run-baseline --policy oracle-lite --scenarios scenarios/mvp --out runs/baselines/oracle_lite_v0.4
python -m lexcapital publish-check --scenarios scenarios/mvp --out audits/publish_mvp_v0.4
pytest -q
```
